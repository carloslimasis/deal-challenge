This is a deals application.

## Requirements

* Maven
* Java 8
* Docker
* Docker Compose
* Thymeleaf
* Spring Boot

## How It Works

Run the command

```sh
$ ./run.sh
```

Then, the Deals App can be accessed on:

```
http://localhost:8086
```
